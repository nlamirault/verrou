# Verrou

[![License Apache 2][badge-license]](LICENSE)

A i3lock wrapper.

## Usage

```bash
$ ./verrou -h
Verrou - v0.1.0
  -dry-run
    	Create background image and not lock screen
  -version
    	print version and exit
```
